# Logo App de Animais — versão iOS Glass

Este pacote está preparado para uma Web App / PWA com visual mais próximo de ícone iOS:
- cantos arredondados;
- brilho/glass no topo;
- reflexo diagonal subtil;
- profundidade e contorno premium;
- ícones em vários tamanhos.

## Ficheiros principais

- `logo-app-animais-ios-glass-1024.png` — versão grande.
- `logo-app-animais-ios-glass.png` — versão 512x512.
- `apple-touch-icon.png` — ícone iOS.
- `favicon.ico` — favicon para browser.
- `manifest.webmanifest` — manifesto PWA.
- `icons/` — ícones em vários tamanhos.
- `colocar-no-head.html` — código para colocar no `<head>`.

## Como usar

1. Copia tudo para a raiz do teu projeto.
2. No `index.html`, cola o conteúdo de `colocar-no-head.html` dentro do `<head>`.
3. Publica novamente no GitHub Pages ou no teu servidor.
4. No iPhone, remove o atalho antigo do ecrã inicial e adiciona outra vez para aparecer o novo ícone.
