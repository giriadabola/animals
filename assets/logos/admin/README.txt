# Pacote Web App / PWA — Ícone Administrativo

Inclui:
- `logo-admin-fundo-transparente.png`
- `logo-admin-512.png`
- `logo-admin-1024.png`
- `favicon.ico`
- `apple-touch-icon.png`
- `manifest.webmanifest`
- pasta `icons/`
- `colocar-no-head.html`

## Como usar
1. Copia todos estes ficheiros para a raiz do teu projeto.
2. Cola o conteúdo de `colocar-no-head.html` dentro do `<head>` do teu HTML.
3. Publica o projeto.
4. Se estiveres a testar no iPhone, apaga o atalho antigo do ecrã inicial e adiciona novamente.
